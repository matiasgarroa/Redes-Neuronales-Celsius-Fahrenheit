# Redes Neuronales - Celsius a Fahrenheit

Este proyecto utiliza una red neuronal simple para realizar la conversión de grados Celsius a grados Fahrenheit.

## Requisitos

- Numpy
- Tensorflow
- Matplotlib

## Uso

Luego de ejecutar todo el código podemos usar la funcion celsius_a_fahrenheit(celsius_prediccion), donde la variable celsius_prediccion es de tipo float, para hacer sus propias predicciones

## Desarrollo

Preparar el entorno de desarrollo: Instalamos las herramientas necesarias: Numpy, Tensorflow y Matplotlib.

Preparar los datos: Creamos un conjunto de datos que contenga valores de temperatura en grados Celsius y su equivalente en grados Fahrenheit.

Definir el modelo de la red neuronal: Decidimos la arquitectura de la red neuronal y definimos sus capas y parámetros.

Entrenar la red neuronal: Utilizamos el conjunto de datos para entrenar la red neuronal y ajustamos los parámetros para obtener un buen rendimiento.

Validar el modelo: Evaluamos el rendimiento del modelo con un conjunto de datos de prueba y lo comparamos con los resultados esperados.

Mejorar el modelo: Si es necesario, mejoramos el modelo ajustando los parámetros para obtener un mejor rendimiento.

Utilizar el modelo: Una vez que hemos obtenido un buen rendimiento en el conjunto de datos de prueba, podemos utilizar el modelo para realizar conversiones de temperatura.
